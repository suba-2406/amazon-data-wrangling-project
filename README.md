# Amazon Data Wrangling Project

## Project Objective
This project focuses on cleaning, transforming, and preparing raw Amazon product data for analysis using Python and Pandas.

## Dataset Overview
The dataset contains Amazon product information including:
- Product names
- Categories
- Prices
- Ratings
- Reviews
- Discounts

## Tools & Technologies
- Python
- Pandas
- NumPy
- VS Code
- GitHub

## Data Cleaning Tasks Performed
- Handled missing values
- Removed special characters
- Converted text columns into numerical datatypes
- Cleaned price and percentage columns
- Feature engineering
- Created analysis-ready dataset

## Feature Engineering
Created new columns:
- main_category
- savings_amount

## Files Included
- Raw dataset
- Cleaned dataset
- Data cleaning script
- Data dictionary
- README documentation

## Output
Generated a clean and analysis-ready Amazon dataset for future analytics and visualization.
